
1. Guía de Inicialización del Proyecto (Paso a Paso)
Sigue estos comandos en tu terminal dentro de la carpeta raíz de tu proyecto para dejar el entorno configurado y listo para programar:

Paso 1: Crear el entorno virtual (Venv)
Es una regla de oro en proyectos profesionales aislar las librerías para evitar conflictos con el sistema operativo.

Bash
python -m venv venv
Paso 2: Activar el entorno virtual
En Windows:

Bash
venv\Scripts\activate
En Linux/macOS:

Bash
source venv/bin/activate
Paso 3: Actualizar el gestor de paquetes e instalar dependencias
Bash

python -m pip install --upgrade pip
pip install -r requirements.txt
